Why you can trust this
This credential was verified inside an Intel TDX hardware enclave — a sealed computer that nobody can look inside, not even the server operator. Three independent guarantees protect the result:
1. Source is real
Data was fetched directly from the authoritative government registry inside the enclave via TLS with a pinned certificate fingerprint — not from a document the user uploaded.
2. Nothing was changed
The certificate is signed with a key that only exists inside the enclave. If anyone modifies the data after issuance, the signature breaks.
3. Proof lives on-chain
The certificate hash is stored on Base (Coinbase L2) permanently. Even if this server disappears, the proof survives on the blockchain.
